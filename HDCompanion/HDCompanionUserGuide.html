Mastering the Companion

The HelpDesk Companion is a unified management console designed to eliminate tool-switching. This guide provides a granular breakdown of every operational feature, from Active Directory automation to remote hardware management.

1. Central Dashboard & Monitoring

Account Lockout Grid

The primary ListView (lvData) automatically tracks locked accounts in the pscu.local domain.

  • • Color Coding: Online/Offline status is visually indicated based on the configured theme.
  • • Multi-Select: Highlight multiple rows to perform bulk unlocks or refreshes.
  • • Sorting: Click any column header to sort by Username, Lockout Time, or Computer.

Automation Sidebar

  • • Auto-Refresh: Choose intervals from 30s to 30m. The app background-queries AD without freezing the UI.
  • • Auto-Unlock: When enabled, the app identifies locked accounts and attempts resolution every 5 minutes (300s).
  • • Email Alerts: Routes notifications to the configured SMTP list (tdawsey, bhigginbotham, adaigle) upon lockout detection.

Global Search

The search bar (txtSearch) is an intelligent entry point.

  • • Context Awareness: Automatically detects if you've typed a username or a computer name.
  • • AD Query: Fetches object properties directly from Active Directory, even if the account isn't locked.
  • • Pause Logic: Background timers pause during active searches to prevent data overwriting.

2. Right-Click Functionality (Deep-Dive)

Core AD Tools
  • Unlock Account: Executes Unlock-ADAccount.
  • Reset Password: Opens the overlayReset. Includes "Strong Password Generator" and "Must change at next logon" enforcement.
  • Group Membership: Opens a dedicated window (GroupMembership.xaml) listing all security and distribution groups.
  • View Details: Dumps all raw AD properties into the overlayDetails window for deep inspection.
Remote Support
  • Manage Computer: Remote process and service management.
  • Find Lockout Source: Scans Domain Controllers for event IDs identifying the specific machine causing the lockout.
  • Active Sessions: Lists users currently logged into the target workstation via WMI.
  • GPResult: Triggers a remote gpresult /r to verify policy application.
Integrations & Power
  • Freshservice Record: Opens the technician's browser directly to the Freshservice ticket or asset URL.
  • Get Uptime: Calculates system boot time to determine if a restart is required.
  • Power Menu: Sub-menu containing verified Restart and Shutdown commands using Stop-Computer.

3. Multi-Threaded Printer Management

Accessed via Right-Click > Printer Management, this module spawns a standalone 64-bit PowerShell process (PrinterManager.ps1) to ensure maximum compatibility with Windows printer drivers.

Diagnostic Functions
  • Restart Spooler Remotely stops and starts the Spooler service to clear stuck jobs.
  • View Details Displays driver version, port name, and sharing status for any selected printer.
Deployment Functions
  • Remote Manifest: Queries the target machine for existing drivers to allow instant installation.
  • INF Staging: Upload a local .inf file to the remote PC's staging area for new hardware support.
  • Local Mirror: Copies a printer driver already installed on your computer to the remote user's computer.

4. Auditing & History

Live Event Log (RichText)

The txtLog box provides real-time feedback with color-coded status levels:

Info: System starts, config loads.
Success: Successful unlocks and password resets.
Error: Failed connections or permission issues.

Historical Log Viewer

Clicking View Logs opens the LogViewer.xaml utility, allowing for long-term audit analysis.

  • • Filtering: Search by Operator (which technician performed the task) or Target User.
  • • Date Range: Narrow down results using the DatePicker controls.
  • • Export: Generate a CSV report for management or compliance reviews.

5. Configuration & Personalization

The app behavior is controlled by HDCompanionCfg.json. You can customize the following via the Settings gear icon:

Technical Settings
  • LogRetentionDays: 90
  • LogDirectoryUNC: \\vm-isserver\Toolkit\...
  • FilteredUsers: ["guest", "support", "admbrian"]
Theming Engine

The app supports dynamic theme switching. Colors are injected into XAML as DynamicResource objects.
